# Cashier

This is a small Node.js app that stores entries in [entries.json](entries.json).

## Files
- [index.html](index.html)
- [style.css](style.css)
- [app.js](app.js)
- [server.js](server.js)
- [entries.json](entries.json)

## Behavior
- Select profile: Mehul or Akhilesh.
- Fill Name, a 10-digit Phone number, Payment Mode, and Paid.
- Data saves only when you click **Save Entry**.
- Saved records are written to `entries.json` in this folder.

## Run Locally
1. Install Node.js.
2. Open this folder in a terminal.
3. Run `npm start`.
4. Open `http://localhost:3000`.

## Deploy
This app is ready for deployment on platforms that support Node.js apps, such as Render, Railway, or any VPS.

## Note
If you want to keep the data file, deploy the whole folder including `entries.json`.
